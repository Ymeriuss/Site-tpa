Introduction

Le but de ce site web sera de présenter les différents éléments importants à prendre en compte si vous voulez devenir pompier.
Ces informations seront réparties en plusieurs points vous permettant une meilleure compréhension de ce site.


Il est important de noter que ces informations viennent de différentes sources et que le but de ce site est de les regrouper pour vous.


Avant de vous expliquez comment devenir Pompier, il est important de vous informer sur les différentes missions de ceux-ci. Voici un petit aperçu :

  • La lutte contre les incendies
  • Le service ambulancier dans les situations d'urgence
  • Lors des accidents de la circulation, pour désincarcérer les per-sonnes coincées dans des véhicules
  • Pour dégager la voie publique après un accident grave
  • En cas d'inondations, explosions, effondrements, etc.
  • Lors de légers problèmes courants : personnes bloquées dans des ascenseurs, etc.

Ce sont toutes ces missions qu'un Pompier doit être capable de réaliser. Contrairement à ce que l'on croit, un pompier ne lutte pas juste contre des incendies ou ne sauve pas juste des chats dans les arbres. Les pompiers font face à énormément de situations différentes.


Nous espérons qu'avec ce site, vous comprendrez mieux comment devenir pompier. Nous espérons que vous vous verrez que ce n'est pas qu'une simple professsion mais un mode de vie à part entière. Mais surtout que grâce à nous vous deviendrez les héros de demain.
